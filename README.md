# Technical Specification:

## 1. System Overview
* **System Basis:** Gridfinity Standard (42mm pitch)
* **Total Height:** 179.75mm (25U)
* **Total Footprint:** 84mm x 126mm (2x3 Gridfinity Units)
* **Weight Capacity:** No more than 200 gms

## 2. Geometric Dimensions (Side Profile)
* **Base Section:**
    * **Depth:** 126mm (Total base length)
    * **Height:** 21.00mm (Standard 3U vertical bin height)
* **Chassis (The Spine):**
    * **Profile Type:** Robust C-Frame/Spine
    * **Position:** Anchored to the rear-most 42mm (1 Unit) of the base
* **Receptacle Interfaces:**
    * **Base Receptacle:** Integrated 2x2x1 Gridfinity (Front-loaded)
    * **Hanging Receptacle:** 2x2x1 Gridfinity, angled at exactly 45° from the horizontal plane
    * **Top Receptacle:** 2x1x1 Gridfinity Modular Space

## 3. Engineering Details
* **Alignment:** The center of the Top receptacle and the center of the Hanging receptacle must lie on the same vertical axis passing through the center of the Front 2×2 baseplate (Center-of-Gravity optimization).
* **Assembly Mechanics:** Single Piece Print or Vertically Two-Piece Cut.
* **Tolerances:** Dimensional tolerances = ±0.5mm.
* **Retention Details:** Integrated pockets for Gridfinity magnets.
* **Material & Print Settings:** PETG+ 3D printed for light receptacle or Aluminum CNC.
